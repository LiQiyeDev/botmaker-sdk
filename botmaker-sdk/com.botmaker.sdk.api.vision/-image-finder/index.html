ImageFinder

open class ImageFinder

Single-frame image lookup: does this template appear right now, and where?

Every matcher takes a CaptureSource — one of a com.botmaker.sdk.api.capture.Window, a monitor, or the whole desktop — so a search can be pinned to a window or a single screen and still return absolute, clickable coordinates. A search region is expressed as a region of a source, not a separate parameter. The no-source overloads default to the whole desktop.

Besides find/findAll/findAny this class owns the lambda control-flow helpers (whileFind, untilFind, ifFind) — each is one capture that hands the matched MatchResult to your action.
