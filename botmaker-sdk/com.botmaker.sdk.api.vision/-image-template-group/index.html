class ImageTemplateGroup : Record

An ordered, non-empty collection of ImageTemplates treated as a single value.

Lets the finder/clicker act on several templates at once: find/click over a group keep first-match semantics (the first template, in order, that clears the confidence threshold wins — cheap short-circuit), while findBest / clickBest evaluate every template and pick the single highest-scoring match. A group also serves as the "good" or "bad" set for the compare API (findCompare).

Immutable: the backing list is copied and unmodifiable.
