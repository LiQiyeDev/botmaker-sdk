ImageClicker

open class ImageClicker

Locate a template and click it. Every method mirrors ImageFinder: a whole-desktop default plus a CaptureSource form (window / monitor / desktop, optionally narrowed with region), so a click can be pinned to a specific surface. The template is located within the source; the click lands at the resulting absolute screen coordinate.

Every method in this class also updates VisionContext for the current thread, enabling access to the most recent match via getLastMatch.
