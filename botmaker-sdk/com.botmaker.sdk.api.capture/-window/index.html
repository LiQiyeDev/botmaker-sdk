Window

open class Window : CaptureSource

A specific application window the bot can target — capture it, click inside it, and manage it (focus / move / resize). Because Window is a CaptureSource, it can be passed straight into the vision layer (e.g. ImageFinder.find(template, window)) so templates are matched and clicked in the window's own coordinate space — surviving window moves, focus changes and multi-monitor layouts.

The underlying native handle is kept opaque; obtain a Window via the static factories.
