Monitor

class Monitor : CaptureSource

A single monitor (0-based index into the OS screen-device list) as a CaptureSource, so a bot can match against just one screen on a multi-monitor desktop. An out-of-range index falls back to the whole desktop's pixels. Prefer the factory monitor over constructing this directly.
