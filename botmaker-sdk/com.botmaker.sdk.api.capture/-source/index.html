class Source

The SDK's global, ambient capture source — the "where" that every no-source vision and mouse call looks at. This lets bots read cleanly: ImageFinder.find(button) instead of threading a CaptureSource through every call.

On first use the current source initialises to the project default source (as configured in Studio and baked into the generated bot), falling back to the whole Desktop when none is configured. Override it at runtime with set — for example to point the whole bot at a game Window once, up front — and every subsequent no-source call follows until it is changed again.

Methods that do take an explicit CaptureSource always use that argument and ignore this global; the global only fills in the no-source overloads.
