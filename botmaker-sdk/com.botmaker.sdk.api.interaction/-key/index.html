Key

enum Key

OS-neutral keyboard keys for Keyboard. Each constant carries both an X11 keysym (Linux) and a Windows virtual-key code; nativeCode resolves to the right one for the current platform so bot code never hard-codes platform key codes.

Letter keys map to the physical key (lowercase keysym on Linux, VK letter on Windows); use type when you need shifted/uppercase characters produced for you.
