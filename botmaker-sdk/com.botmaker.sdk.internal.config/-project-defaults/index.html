ProjectDefaults

class ProjectDefaults

Reads the optional per-project defaults that Studio bakes into a generated bot as a classpath resource ({@value #RESOURCE}). Everything here is best-effort: a missing file, missing key, or unparseable value yields null so callers fall back to their own defaults (the whole Desktop for the source, native pixels for the resolution). Loaded once and cached.

Recognised keys:

  • capture.source — desktop | monitor:<index> | window:<titleSubstring>
  • capture.width / capture.height — the resolution templates were authored at
