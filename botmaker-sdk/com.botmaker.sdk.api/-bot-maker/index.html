BotMaker

open class BotMaker

Single entry point for basic console IO from a bot.

Print and read are exposed as static methods so user code never has to manage a Scanner or System.out directly — the visual Print / Read blocks compile down to BotMaker.print(...) and BotMaker.readX() calls.
